Publish to the source

Commits the edited package straight into extra-debs/. The build then indexes it and it appears in Sileo — no computer needed.

About the token. It grants write access to that repository, so use a fine-grained token limited to this one repo with only Contents: read and write, and nothing else. Remembering it stores it in this browser — leave that off on a shared device. Revoke it any time from GitHub settings.

Only the control metadata is rewritten. The package payload is copied through untouched, so nothing you edit here can break the tweak itself.

Reading and editing happen entirely on this device. Nothing leaves it unless you press Publish, which sends the package to your own repository and nowhere else.
